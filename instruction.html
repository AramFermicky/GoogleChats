Выбор типа подключения

GoogleChats поддерживает два типа P2P подключения:

  • Wi-Fi Direct: Прямое подключение в локальной сети
  • Bluetooth: Беспроводное подключение на расстоянии до 10 метров

Выберите подходящий тип в боковой панели приложения.

devices

Иконки подключения в боковой панели

2

Обмен идентификаторами

Каждому устройству присваивается уникальный ID. Для подключения:

  1. Скопируйте свой ID из поля "ВАШ ID"
  2. Передайте его собеседнику (QR-код, сообщение, устно)
  3. Вставьте ID собеседника в поле "ПОДКЛЮЧИТЬСЯ К"
  4. Нажмите кнопку "Подключиться"
googlechats_abc123xyz

Пример идентификатора устройства

3

Общение в реальном времени

После успешного подключения:

  • Вводите сообщения в нижнем поле
  • Нажимайте Enter или иконку отправки
  • Сообщения передаются напрямую между устройствами
  • История хранится только на вашем устройстве

Статус подключения отображается в правом верхнем углу.

security Безопасность

Все сообщения шифруются end-to-end. Данные не проходят через серверы Google. Соединение устанавливается напрямую между вашими устройствами.
